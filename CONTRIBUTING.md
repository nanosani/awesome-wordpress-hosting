# Contributing to Awesome WordPress Hosting

Thank you for helping improve this list! All genuine contributions are welcome.

## How to Add an Item

1. Fork this repository.
2. Edit `README.md` and add your item in the correct section.
3. Use this format: `[Name](URL) — One-sentence description.`
4. Keep items in **alphabetical order** within their section.
5. Submit a pull request with a clear title (e.g., `Add Hetrix Tools to Monitoring section`).

## Quality Standards

- The plugin/tool/service must be **actively maintained** (updated within the last 12 months).
- Services must have a **publicly accessible website** with clear pricing or free tier.
- WordPress plugins must have at least **500 active installs** or be a well-known commercial product.
- No affiliate or referral links.
- No self-promotion unless the tool has meaningful community adoption.

## What Not to Add

- Abandoned or deprecated tools.
- Tools with known unpatched security vulnerabilities.
- Duplicate entries — check the list before adding.
- Items that don't fit the hosting/reseller focus of this list.

## Reporting Issues

Found a broken link, outdated entry, or incorrect description? Open an issue or submit a PR with the fix directly.
